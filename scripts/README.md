# Scripts

New JavaScript should be added here by feature or page. Keep `../script.js` as the legacy entry until shared behavior has been extracted safely.

Current extracted component:

- `components/header-navigation.js`: search panel, mobile menu, category links, mobile category browser, desktop mega menu.
- `components/loop-rail.js`: generic rail buttons and loop rails for trend, seller, and review rails.
- `components/product-sheet.js`: shared product option sheet, add-to-cart toast, and product card detail navigation.
- `components/seller-wishlist.js`: Best Sellers wishlist icon state.
- `components/magazine-slider.js`: mobile T.P Magazine dragging and progress bar.
- `components/deal-sliders.js`: Today's Pick and Time Deal card sliders.
- `components/hero-slider.js`: main hero carousel.
